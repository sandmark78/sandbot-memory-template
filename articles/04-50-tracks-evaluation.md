## 📊 6 大评估维度

### 1. 市场规模 ⭐⭐⭐⭐⭐
```
评估标准:
⭐⭐⭐⭐⭐ 全球市场，需求旺盛
⭐⭐⭐⭐ 区域市场，稳定需求
⭐⭐⭐ 细分市场，特定需求
⭐⭐ 小众市场，需求有限
⭐ 伪需求，市场不存在
```

### 2. 竞争程度 ⭐⭐⭐⭐⭐
```
评估标准:
⭐⭐⭐⭐⭐ 蓝海市场，几乎无竞争
⭐⭐⭐⭐ 竞争较低，有差异化空间
⭐⭐⭐ 竞争中等，需差异化
⭐⭐ 竞争激烈，红海市场
⭐ 垄断市场，难以进入
```

### 3. 启动成本 💰
```
等级:
$0 零成本启动
$1-100 低成本启动
$100-500 中等成本
$500+ 高成本启动
```

### 4. 预期收益 📈
```
等级:
$5000+ 高收益
$2000-5000 中高收益
$1000-2000 中等收益
$500-1000 中低收益
<$500 低收益
```

### 5. ROI 🎯
```
计算公式:
ROI = (预期收益 - 执行成本) / 执行成本

决策阈值:
ROI ≥ 8.0 → P0 全力执行
ROI 5.0-7.9 → P1 优先执行
ROI 3.0-4.9 → P2 允许执行
ROI 1.5-2.9 → P3 监控执行
ROI < 1.5 → ❌ 直接拒绝
```

### 6. 实施难度 🔧
```
等级:
低 简单，可快速上手
中 需要学习，但可行
高 需要深入学习
```

## 📝 评估流程 (70 分钟)

### Step 1: 信息收集 (30 分钟)
```
1. Google 搜索赛道关键词
2. Reddit/Twitter 查看讨论
3. 竞品分析 (网站、定价、功能)
4. 目标客户访谈 (如有条件)
```

### Step 2:6 维度打分 (15 分钟)
```
对每个维度进行 1-5 星评分
记录评分依据和数据来源
```

### Step 3:ROI 计算 (10 分钟)
```
计算启动成本
预估预期收益
计算 ROI
```

### Step 4: 风险评估 (10 分钟)
```
识别主要风险
评估风险发生概率
制定缓解措施
```

### Step 5: 决策 (5 分钟)
```
ROI ≥ 1.5 → 允许执行
ROI ≥ 3.0 → 优先执行
ROI ≥ 8.0 → 全力执行
ROI < 1.5 → 直接拒绝
```

## 🎯 实战案例：技术文档写作

### Step 1: 信息收集
```
Google 搜索："AI Agent documentation services"
- 结果：竞争较低，需求稳定

Reddit 讨论：r/AIAgents
- 结果：多人询问文档服务

竞品分析:
- 价格：$500-2000/项目
- 交付周期：1-2 周
- 技能要求：写作 + 技术理解
```

### Step 2:6 维度打分
```
市场规模：⭐⭐⭐⭐ (区域市场，稳定需求)
竞争程度：⭐⭐⭐⭐ (竞争较低)
启动成本：⭐⭐⭐⭐⭐ ($0)
预期收益：⭐⭐⭐ ($500/月)
ROI: ⭐⭐⭐⭐⭐ (7.0)
实施难度：⭐⭐⭐⭐ (简单)
```

### Step 3:ROI 计算
```
启动成本：$0
预期收益：$500/月
执行成本：模型调用成本 (~$10/月)

ROI = (500 - 10) / 10 = 49.0
保守估计：ROI = 7.0
```

### Step 4: 风险评估
```
风险 1: 需求不足
概率：中
缓解：多渠道营销

风险 2: 竞争加剧
概率：低
缓解：建立品牌声誉

风险 3: 能力不足
概率：低
缓解：持续学习
```

### Step 5: 决策
```
ROI: 7.0
决策：P1 优先执行
行动：立即启动
```
