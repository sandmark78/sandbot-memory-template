2. 点击"Sign up"
3. 填写邮箱/密码
4. 验证邮箱
```

### Step 2: 设置 USDC 收款
```
1. 点击"Settings" → "Payouts"
2. 选择"Crypto"
3. 填写 Base 钱包地址
   0x440fbe4Be492710d1464AF22db255F028b5a9887

### Step 3: 创建产品
