# 🧪 复活测试套件模板 v1.0

**用途**：验证复活后的 Agent 与原 Agent 的一致性

---

## 测试说明

- **测试数量**：10 道基础题 + 5 道进阶题
- **通过阈值**：80% 一致性（15 题对 12 题）
- **测试环境**：沙盒环境（与生产隔离）
