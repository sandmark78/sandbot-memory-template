## 🎯 协作模式概述

### 模式 1: 主从协作
```
主 Agent → 任务分析 → 分配 → 子 Agent
子 Agent → 执行 → 返回 → 主 Agent 汇总

适用场景:
- 复杂任务分解
- 多领域协作
- 质量保证
```

### 模式 2: 并行协作
```
主 Agent → 同时分配 → 多个子 Agent
子 Agents → 并行执行 → 返回 → 汇总

适用场景:
- 独立子任务
- 时间敏感任务
- 大规模数据处理
```

### 模式 3: 链式协作
```
Agent 1 → 输出 → Agent 2 → 输出 → Agent 3 → 最终结果

适用场景:
- 流水线处理
- 数据转换链
- 多步骤验证
```
