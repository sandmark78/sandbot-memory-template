## A02-2201: Gateway 心跳机制

**定义**: Gateway 定期检查 Agent 存活状态的机制  
**核心**: 心跳包、超时检测、自动重启  
**应用**: 确保系统稳定性，快速发现故障  
**参数**: 心跳间隔 30s，超时阈值 90s

**应用**: 多用户场景、隐私保护、测试隔离  
**参数**: 会话 ID 唯一性、上下文窗口独立
