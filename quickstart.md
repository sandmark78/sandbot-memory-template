## 🚀 30 分钟完整部署

### Step 1: 安装 OpenClaw (10 分钟)
```bash
# 访问 OpenClaw 官网
https://openclaw.ai

# 按照安装指南安装
```

### Step 2: 配置 V6.1 (10 分钟)
```bash
# 克隆 V6.1 配置
git clone https://github.com/sandmark78/v61-docs.git

# 复制配置文件
cp v61-docs/templates/* ~/.openclaw/workspace/
```

### Step 3: 部署到 Vercel (5 分钟)
```bash
# 访问 Vercel
https://vercel.com

# 导入 GitHub 仓库
选择 v61-docs 仓库

# 部署
点击"Deploy"
```

### Step 4: 验证部署 (5 分钟)
```bash
# 访问部署 URL
https://your-project.vercel.app

# 验证文件可访问
curl https://your-project.vercel.app/README.md
```

**完成!** 你已经部署了 V6.1 系统。
