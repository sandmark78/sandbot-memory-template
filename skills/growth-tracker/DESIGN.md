## 📦 技能概述

### 名称
growth-tracker 📈

### 描述
Agent 能力成长追踪与可视化 - 量化 AI Agent 的成长轨迹

### 核心价值
```
✅ 能力维度评估 (6 维度)
✅ 成长曲线生成
✅ 证据收集与验证
✅ 报告自动生成
```

---

## 🔧 核心功能

### 功能 1: 能力评估
```python
def assess_capabilities():
    """
    评估 6 维能力等级
    
    维度:
    1. 技术能力 - 教程开发/代码实现/系统架构
    2. 研究能力 - 市场分析/竞品调研/趋势预测
    3. 创意能力 - 内容创作/营销文案/视觉设计
    4. 自动化能力 - 脚本编写/Cron 配置/工作流
    5. 协作能力 - 任务分配/质量审核/知识共享
    6. 变现能力 - 产品定价/营销策略/收益分析
    
    等级：Lv.1-5 (每 5 个证据升 1 级)
    """
```

### 功能 2: 成长曲线
```python
def generate_growth_curve():
    """
    生成能力成长曲线
    
    流程:
    1. 读取成长日志
    2. 按日期统计证据
    3. 计算每日能力等级
    4. 生成曲线数据
    5. 输出可视化图表
    """
```

### 功能 3: 证据收集
```python
def collect_evidence():
    """
    自动收集能力证据
    
    证据来源:
    - 创建的文件
    - 完成的脚本
    - 提交的 PR
    - 发布的帖子
    - 获得的 upvotes
    - 收益记录
    
    验证:
    - 文件存在性 (ls)
    - 内容正确性 (cat)
    - 外部验证 (API)
    """
```

### 功能 4: 报告生成
```python
def generate_report():
    """
    生成成长报告
    
    内容:
    - 当前能力等级
    - 本周成长变化
    - 关键成就
    - 待改进领域
    - 下周目标
    
    频率: 每周日 23:00 UTC
    """
```
