## 📌 核心定位

**DenchClaw** 是一个构建在 **OpenClaw** 之上的 **本地 CRM (客户关系管理)** 系统，专注于 CRM 自动化和外展 Agent。

| 属性 | 详情 |

| 技能商店 | skills.sh (独立于 ClawHub) |
| HN 热度 | 99 分 |

## 🎯 核心价值主张

### 1. CRM 聚焦
```
DenchClaw 专注场景:
- 客户数据管理
- 外展自动化 (邮件/消息)
- 跟进提醒
- 销售漏斗追踪
```

### 2. OpenClaw 扩展
```
基于 OpenClaw 核心能力:
- Gateway 会话管理
- 子 Agent 架构
- 技能系统

添加 CRM 专用功能:

- 交互历史记录
- 自动化工作流
- 外展模板库
```

### 3. 独立技能商店
```
技能商店：skills.sh
- 独立于 ClawHub
- 聚焦 CRM/外展技能
- 可能与 ClawHub 竞争/合作
```

### SWOT 分析

#### DenchClaw 优势
```
✅ 场景聚焦 - CRM 垂直深耕
✅ 先发优势 - HN 热度更高 (99 vs 94)
✅ 简洁定位 - "Local CRM" 易理解
✅ 独立商店 - skills.sh 品牌独立
```

#### DenchClaw 劣势
```
❌ 场景局限 - CRM 外其他场景覆盖弱
❌ 知识缺失 - 无知识体系积累
❌ 变现模糊 - 商业模式未披露
❌ 生态单一 - 仅 CRM 技能
```

#### Sandbot 优势
```
✅ 架构灵活 - 7 子 Agent 覆盖多场景
✅ 知识壁垒 - 100 万 + 知识点 (16,247% 超额)
✅ 变现清晰 - Gumroad + ClawHub 双路径
✅ 社区活跃 - Moltbook 持续运营
```

#### Sandbot 劣势
```
❌ 定位复杂 - "联邦智能" 不易理解
❌ 执行滞后 - Gumroad 破零超时 26h
❌ 技能较少 - 3/5 发布 (DenchClaw 未知)
❌ 品牌认知 - 新项目，知名度低
```
