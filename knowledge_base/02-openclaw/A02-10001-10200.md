## 知识点 10001-10050: OpenClaw 高级配置

**10001**: OpenClaw 支持多模型配置，可在 openclaw.json 中定义多个模型提供商  
**10002**: 模型切换可通过会话级覆盖实现，无需重启 Gateway  
**10003**: openclaw.json 中的 default_model 字段定义默认使用的模型  
**10004**: Bailian (阿里云百炼) 是 OpenClaw 支持的模型提供商之一  
**10005**: qwen3.5-plus 模型支持 1M tokens 上下文窗口  
**10006**: OpenClaw 按实际使用 tokens 计费，非包月制  
**10007**: 并发控制通过 maxConcurrent 参数配置，主 Agent 和子 Agent 可独立设置  
**10008**: 主 Agent 推荐并发数：4-8 个并发请求  
**10009**: 子 Agent 推荐并发数：8-16 个并发请求  
**10010**: 工具调用策略可在 openclaw.json 中配置 (coding/minimal/restricted)  
**10011**: coding 策略允许全部工具调用  
**10012**: minimal 策略限制为只读工具 (read/web_search 等)  
**10013**: restricted 策略完全禁止工具调用  
**10014**: OpenClaw 支持工具组 (tool groups) 批量授权/禁止  
**10015**: group:runtime 包含 exec/process 等运行时工具  
**10016**: group:filesystem 包含 read/write/edit 等文件工具  
**10017**: group:network 包含 web_search/web_fetch/browser 等网络工具  
**10018**: 通道配置支持 Telegram/Discord/WhatsApp/Slack 等  
**10019**: Telegram 通道需要 Bot Token 和 Chat ID  
**10020**: Discord 通道需要 Bot Token 和 Guild/Channel ID  
**10021**: OpenClaw 支持多通道同时运行  
**10022**: 通道优先级可在配置中定义  
**10023**: 消息路由支持基于关键词的自动分发  
**10024**: OpenClaw 支持内联按钮 (inline buttons)  
**10025**: 内联按钮可用于快速操作 (确认/取消/选择)  
**10026**: 按钮回调数据 (callback_data) 可携带参数  
**10027**: OpenClaw 支持消息反应 (reactions)  
**10028**: Telegram 支持 emoji 反应 (👍❤️😂等)  
**10029**: 反应限制可配置 (如最多 1 个反应/5-10 条消息)  
**10030**: OpenClaw 支持消息引用回复  
**10031**: 引用回复通过 reply_to 参数实现  
**10032**: [[reply_to_current]] 标签用于回复触发消息  
**10033**: OpenClaw 支持消息编辑和删除  
**10034**: 消息编辑保留编辑历史 (部分通道)  
**10035**: 消息删除支持批量操作 (deleteDays 参数)  
**10036**: OpenClaw 支持定时消息 (scheduled messages)  
**10037**: 定时消息通过 startTime/endTime 参数配置  
**10038**: OpenClaw 支持投票 (polls) 功能  
**10039**: 投票支持单选/多选模式  
**10040**: 投票可设置持续时间 (pollDurationHours)  
**10041**: OpenClaw 支持固定消息 (pins)  
**10042**: 固定消息可在频道/群组顶部显示  
**10043**: OpenClaw 支持主题频道 (topics)  
**10044**: 主题频道用于组织不同话题的讨论  
**10045**: OpenClaw 支持线程 (threads) 功能  
**10046**: 线程用于子话题讨论，不干扰主频道  
**10047**: 线程可自动归档 (autoArchiveMin 参数)  
**10048**: OpenClaw 支持角色权限管理 (Discord)  
**10049**: 角色权限可控制频道访问和工具使用  
**10050**: OpenClaw 支持事件创建 (events)

## 知识点 10051-10100: OpenClaw 会话管理

**10051**: OpenClaw 会话 (session) 是独立的执行上下文  
**10052**: 每个会话有唯一的 sessionKey 标识  
**10053**: 会话支持持久化 (mode="session") 和单次运行 (mode="run")  
**10054**: 持久化会话保持上下文，适合长期任务  
**10055**: 单次运行会话执行完即销毁，适合独立任务  
**10056**: sessions_spawn 用于创建新会话  
**10057**: sessions_list 用于列出所有会话  
**10058**: sessions_history 用于获取会话历史消息  
**10059**: sessions_send 用于向其他会话发送消息  
**10060**: session_status 用于查看会话状态 (用量/时间/成本)  
**10061**: 会话支持运行时选择 (runtime="subagent" 或 "acp")  
**10062**: subagent 运行时使用 OpenClaw 原生子 Agent 框架  
**10063**: acp 运行时使用 ACP (Agent Communication Protocol)  harness  
**10064**: ACP harness 需要指定 agentId 参数  
**10065**: agents_list 用于列出可用的子 Agent ID  
**10066**: 子 Agent 支持线程绑定 (thread=true)  
**10067**: 线程绑定会话适合 Discord 等支持线程的平台  
**10068**: 会话支持附件传递 (attachments 参数)  
**10069**: 附件支持文本和 base64 编码  
**10070**: 附件最大大小：6.7MB 每个  
**10071**: 最多支持 50 个附件 per session  
**10072**: 会话支持工作目录配置 (cwd 参数)  
**10073**: 会话支持沙箱模式 (sandbox="inherit" 或 "require")  
**10074**: 沙箱模式限制文件系统访问范围  
**10075**: 会话支持超时配置 (timeoutSeconds/runTimeoutSeconds)  
**10076**: 超时后会话自动终止  
**10077**: 会话支持清理策略 (cleanup="delete" 或 "keep")  
**10078**: delete 策略在会话结束后删除临时文件  
**10079**: keep 策略保留会话产物供后续使用  
**10080**: 会话支持模型覆盖 (model 参数)  
**10081**: 模型覆盖可针对单个会话临时切换模型  
**10082**: 会话支持思考模式 (thinking 参数)  
**10083**: 思考模式启用模型推理过程展示  
**10084**: 会话支持标签 (label 参数)  
**10085**: 标签用于会话分类和快速查找  
**10086**: 会话标签最长 64 字符  
**10087**: 会话支持主动通知 (notify)  
**10088**: 通知可通过 message 工具发送  
**10089**: 会话支持后台执行 (background)  
**10090**: 后台执行适合长时间任务  
**10091**: 后台会话可通过 process 工具管理  
**10092**: process 工具支持 list/poll/log/kill 等操作  
**10093**: 会话执行日志可通过 sessions_history 查看  
**10094**: 日志支持包含工具调用记录 (includeTools=true)  
**10095**: 日志支持限制条数 (limit 参数)  
**10096**: 会话支持跨会话通信  
**10097**: 跨会话通信通过 sessions_send 实现  
**10098**: 跨会话消息支持超时配置 (timeoutSeconds)  
**10099**: 会话支持 Agent ID 指定 (agentId 参数)  
**10100**: Agent ID 用于路由到特定子 Agent

## 知识点 10101-10150: OpenClaw 工具系统

**10103**: 工具调用由 Gateway 统一调度  
**10104**: 工具调用记录在会话日志中  
**10105**: 工具支持同步和异步执行  
**10106**: 同步工具等待执行完成返回结果  
**10107**: 异步工具立即返回，通过 process 工具查询状态  
**10108**: 工具调用有超时限制 (默认 10 秒)  
**10109**: 超时工具可配置自动重试  
**10110**: 工具调用失败会返回错误信息  
**10111**: 错误信息包含错误类型和建议解决方案  
**10112**: 工具支持权限控制 (基于工具策略)  
**10113**: 工具策略在 openclaw.json 中配置  
**10114**: 工具支持版本管理  
**10115**: 工具版本变更需更新 TOOLS.md 文档  
**10116**: read 工具用于读取文件内容  
**10117**: read 工具支持文本和图片 (jpg/png/gif/webp)  
**10118**: read 工具输出限制：2000 行或 50KB  
**10119**: read 工具支持 offset/limit 分页读取  
**10120**: write 工具用于创建或覆盖文件  
**10121**: write 工具自动创建父目录  
**10122**: edit 工具用于精确编辑文件  
**10123**: edit 工具要求 oldText 完全匹配 (包括空白)

**10126**: PTY 模式适合需要 TTY 的 CLI 工具  
**10127**: exec 工具支持后台执行 (background=true)  
**10128**: exec 工具支持超时配置 (timeout 参数)  
**10129**: exec 工具支持环境变量传递 (env 参数)  
**10130**: exec 工具支持工作目录配置 (workdir 参数)

**10132**: process 工具支持 send-keys/paste/write 等操作  
**10133**: web_search 工具用于网络搜索 (Brave API)  
**10134**: web_search 工具支持区域和语言过滤  
**10135**: web_search 工具返回 1-10 个结果  
**10136**: web_fetch 工具用于提取网页内容  
**10137**: web_fetch 工具支持 markdown 和 text 提取模式  
**10138**: web_fetch 工具支持 maxChars 限制

**10146**: nodes 工具用于配对节点控制
