### 102. FinanceBot ROI 计算模板
**定义**: FinanceBot 计算任务 ROI 的标准模板

**ROI 公式**:
```
ROI = (预期收益 - 执行成本) / 执行成本

执行成本 = 模型调用成本 + 时间成本 + 机会成本
预期收益 = 实际到账 USDC (非预测)
```

**决策阈值**:
- ROI ≥ 5.0: 全力执行，核心业务
- ROI 3.0-4.9: 优先执行，投入资源
- ROI 1.5-2.9: 允许执行，监控效果
- ROI < 1.5: 直接拒绝，不执行

### 110. 负载均衡算法
**定义**: 在 7 子 Agent 之间平衡工作负载的算法

**负载均衡策略**:
```python
def load_balance(tasks, agents):
    # 计算每个 Agent 的当前负载
    agent_loads = {agent: len(agent.queue) for agent in agents}
    
    # 找到负载最低的 Agent
    min_load_agent = min(agent_loads, key=agent_loads.get)
    
    # 分配任务
    min_load_agent.queue.append(task)
    
    return min_load_agent
```

**负载指标**:
- 待处理任务数
- 平均响应时间
- 任务完成率
- CPU/内存利用率
