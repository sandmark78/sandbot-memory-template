## 📊 核心概述

**BitNet 100B** 是微软研究院推出的 1-bit 大语言模型，支持纯 CPU 本地推理，标志着边缘 AI 推理的重大突破。

### 关键技术参数
| 参数 | 数值 | 意义 |
|------|------|------|
| 参数量 | 100B | 与主流大模型相当 |
| 量化精度 | 1-bit | 每个权重仅 1 位 |
| 推理设备 | CPU | 无需 GPU |
| 推理速度 | ~10 tokens/s | 消费级 CPU |
| 内存占用 | ~12GB | 普通 PC 可运行 |

---

## 🔬 技术原理

### 1-bit 量化核心思想
```
传统模型：32-bit 浮点权重 → 100B × 32 bits = 3.2TB
BitNet:    1-bit 三值权重 → 100B × 1 bit  = 12.5GB

压缩比：256:1
```

### 三值表示法
```
权重 ∈ {-1, 0, +1}

优势:
1. 乘法变加法：W × x → sign(W) × |x|
2. 稀疏激活：0 权重直接跳过
3. 位运算加速：XOR + popcount
```

### 训练方法
```
1. 全精度预训练 (32-bit)
2. 渐进式量化 (32→8→4→2→1 bit)
3. 量化感知微调 (QAT)
4. 蒸馏优化 (教师 - 学生)
```

### 部署方案
```bash
# 1. 模型下载
git clone https://github.com/microsoft/BitNet
cd BitNet/models/100B-1bit

# 2. 依赖安装
pip install bitnet-inference torch

# 3. 本地推理
python inference.py \
  --model bitnet-100b-1bit \
  --prompt "Hello, world" \
  --device cpu \
  --threads 8
```
