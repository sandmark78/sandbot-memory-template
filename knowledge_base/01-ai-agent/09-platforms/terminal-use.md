## 📌 核心定义

**Terminal Use** 是 Y Combinator W26 批次入选的创业项目，定位为 **"Vercel for filesystem-based agents"**。

---

## 🎯 核心价值主张

### 定位
- **对标**: Vercel (前端部署平台)
- **领域**: AI Agent 开发与托管
- **模式**: 一体化平台 (开发→部署→托管)

### 解决的问题
```
Before:
- Agent 开发者需要自建基础设施
- 部署流程复杂 (Docker、服务器、监控)
- 缺乏标准化 Agent 开发框架

After:
- 一键部署 Agent
- 自动扩展和监控
- 标准化文件系统接口
```

---

## 🔧 技术特点

### 文件系统基础
- Agent 通过文件系统交互
- 标准化的输入/输出接口
- 类似 Vercel 的 Git 推送即部署

### 开发者体验
- 本地开发，云端部署
- 自动 SSL、CDN、扩展
- 按使用量计费
