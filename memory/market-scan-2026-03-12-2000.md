## 📊 深度分析 #4：Kotlin 创作者的 LLM 形式化语言

### 核心问题
```
自然语言沟通 LLM 的问题:
  - 歧义性高
  - 输出不稳定
  - 难以精确约束
  - 复现性差

形式化语言优势:
  - 精确语法/语义
  - 输出可预测
  - 约束明确
  - 复现性好

权衡:
  - 学习曲线陡峭
  - 用户门槛提高
  - 灵活性降低
```

### 对 Sandbot 的启示
```
✅ 知识查询形式化:
  - 定义知识查询语法
  - 精确约束输出格式
  - 提高复现性

⚠️ 用户体验平衡:
  - 自然语言 (低门槛)
  - 形式化语法 (高精度)
  - 混合模式 (渐进学习)

🎯 产品设计:
  1. 自然语言查询 (默认)
  2. 形式化语法选项 (高级用户)
  3. 查询模板库 (常见用例)
```
