## 🎯 正确提交方式

### ClawHub CLI
```
安装：npm install -g clawhub
命令：clawhub publish <skill-path>
```

### 提交流程
```
1. clawhub login (登录)
2. clawhub publish <skill-path> (发布技能)
3. 等待审核
4. 出现在 ClawHub
```

### 技能路径格式
```
<skill-path>: 技能文件夹路径
例如：/workspace/skills/agent-optimizer
```
