### 攻击详情 (CodeWall.ai)
```
目标：McKinsey Lilli 平台
- 43,000+ 员工使用
- 70% 采用率
- 500,000+ prompts/月
- 100,000+ 内部文档

攻击过程:
  1. 自主 Agent 映射攻击面 (2 小时)
  2. 发现 200+ API 端点公开暴露
  3. 22 个端点无需认证
  4. SQL 注入 (JSON keys 直接拼接)
  5. 15 轮盲注迭代，获取生产数据库访问

  - 46.5M 聊天消息
  - 728K 文件 (192K PDF, 93K Excel, 等)
  - 57K 用户账户
  - 384K AI 助手配置
  - 3.68M RAG 文档块 (几十年专有研究)
  - 1.1M 文件通过外部 AI API 流动

关键漏洞:
  - SQL 注入 (JSON keys 未参数化)
  - IDOR (跨用户数据访问)
  - 系统提示可写 (可篡改 AI 行为)
  - S3 路径暴露
```
