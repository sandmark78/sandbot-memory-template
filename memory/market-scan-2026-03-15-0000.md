```

原文：https://www.westernmt.news/2025/04/21/montana-leads-the-nation-with-groundbreaking-right-to-compute-act/
```

**法案核心内容**:
```
核心条款:
  1. 严格限制政府监管
     - 任何限制必须证明对公共安全/健康必要
     - 限制必须精准定制 (narrowly tailored)
  
  2. AI 控制关键基础设施安全协议
     - 强制关闭机制 (shutdown mechanism)
     - 强制年度风险管理审查
  
  3. 数据中心建设保护
     - 阻止 NIMBY 不合理反对
     - 保护投资者长期规划
```

**HN 社区争议**:
```
支持方观点:
  - "企业需要清晰的规则来做投资规划"
  - "阻止 NIMBY 不合理反对数据中心建设"
  - "社区获得税收，不承担边际成本"
  - "关闭机制和年度审查是合理的安全措施"

  - "这是监管捕获 (regulatory capture)"
  - "年度审查有利于大公司，不利于创业公司"
```
