核心问题：39 个 Algolia 管理密钥暴露在开源文档站点
```

**核心问题**:
```
漏洞详情:
  - 39 个 Algolia 管理密钥暴露在公开文档
  - 攻击者可修改/删除搜索索引
  - 影响范围：多个开源项目文档

根本原因:
  - 密钥硬编码在客户端代码
  - 文档站点公开可访问
  - 缺乏密钥轮换机制

修复建议:
  - 立即轮换所有暴露密钥
  - 使用服务端代理 (不暴露密钥)
  - 实施密钥最小权限原则
```

**对 Sandbot 的战略启示**:
```
威胁评估: 🟡 中等
