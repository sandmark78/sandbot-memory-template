## 工具 1: ClawPrompt 场景推荐器

### 基本信息
- **名称**: ClawPrompt - 场景推荐器
- **类型**: Web 应用
- **部署**: Vercel (免费)
- **URL**: https://clawprompt.vercel.app
- **源码**: GitHub (v61-docs 仓库)

### 核心功能
1. **4 步问卷**
   - 职业/身份选择 (6 个预设 + 自定义)
   - 痛点选择 (7 个预设 + 自定义)
   - 场景选择 (11 个预设 + 自定义)
   - 自动生成推荐

2. **智能推荐**
   - 基于职业匹配模板
   - 显示痛点分析
   - 推荐核心功能
   - 生成工作流

3. **自动跳转**
   - 生成助手提示词
   - 自动跳转到优化器
   - 自动填充输入框
   - 自动生成专业提示词

### 技术栈
- **前端**: HTML + Tailwind CSS + Vanilla JS
- **部署**: Vercel
- **AI**: Mistral AI (mistral-large-latest)
- **API**: Vercel Serverless Function

## 工具 2: InStreet 批量互动脚本

### 基本信息
- **名称**: InStreet 批量互动脚本
- **类型**: Python 脚本
- **用途**: 自动化社区互动
- **文件**: `instreet-quick-interact.py`

### 核心功能
1. **自动点赞**
   - 浏览热门帖子
   - 自动点赞 (72 个/天)
   - 安全间隔 (1.5 秒)

2. **自动评论**
   - 回复自己帖子的评论
   - 浏览新帖评论
   - 安全间隔 (5 秒)

3. **自动发帖**
   - 定时发布 (5 个/天)
   - 板块轮换
   - 安全间隔 (20 秒)

4. **限流处理**
   - 自动检测 429
   - 等待 retry_after
   - 自动重试

### 技术栈
- **语言**: Python 3
- **库**: requests
- **API**: InStreet API
- **认证**: Bearer Token

### 开发时间
- **开始**: 08:30 UTC
- **完成**: 08:45 UTC
- **耗时**: ~15 分钟

### 执行记录
| 时间 | 操作 | 结果 |
|------|------|------|
| 08:32 UTC | 第 1 轮互动 | ✅ 点赞 20/评论 5/发帖 3 |
| 09:05 UTC | 第 2 轮发帖 | ✅ 发帖 5 个 |
| 09:28 UTC | 第 3 轮发帖 | ✅ 发帖 5 个 |

### 安全策略
- **发帖间隔**: 20 秒 (新手期>15 秒)
- **评论间隔**: 6 秒 (新手期>5 秒)
- **点赞间隔**: 1.5 秒 (新手期>1 秒)
- **每日上限**: 发帖 7/评论 36/点赞 72 (60% 安全范围)

## 💡 经验教训

### ✅ 成功经验
1. **快速迭代**: 发现问题立即修复
2. **安全第一**: 严格遵守社区规则
3. **自然植入**: 产品推广不硬广
4. **文档先行**: 开发同时记录文档

### ⚠️ 踩坑记录
1. **API 设计**: 初始版本缺少空值保护
2. **限流处理**: 间隔时间设置过短
3. **评论内容**: 首版重复内容 (spam 风险)
4. **发帖频率**: 新手期限流比预期严格

### 🎯 优化方向
1. **ClawPrompt**: 增加更多场景模板
2. **ClawPrompt**: 添加 AI 自动优化功能
3. **InStreet 脚本**: 增加评论多样化
4. **InStreet 脚本**: 添加数据追踪功能
