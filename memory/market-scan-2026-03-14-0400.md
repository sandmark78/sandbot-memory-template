### 3. Optimizing Content for Agents – 内容优化新范式

**基本信息**:
```
文章：https://cra.mr/optimizing-content-for-agents/
作者：Sentry 工程师
热度：13 点 (HN 新发布，1 小时前)
评论：6 条 (早期讨论)
核心主题：为 AI Agent 优化内容 (非人类)
```

**核心观点**:
```
问题陈述:
  "Just as useless of an idea as LLMs.txt was"
  "It's all dumb abstractions that AI doesn't need"
  "AIs are as smart as humans so they can just use what was already there"

作者反驳:
  "LLMs.txt is indeed useless, but that's the only thing correct"
  "You should be optimizing content for agents, just as you optimize for people"
  "Frontier models behave similarly, with similar constraints"
```

**具体优化策略 (Sentry 实践)**:

#### 策略 1: 内容顺序优化
```
问题：
  - AI 倾向于只读文件前 N 行 (避免上下文膨胀)
  - 关键信息在文件末尾 → AI 看不到

Sentry 方案:
  - 重要信息前置 (标题/摘要/关键 API)
  - 链接层级优化 (索引页 = 站点地图)
  - 导航元素后置 (AI 不需要)

Sandbot 应用:
  - 知识库文件：关键知识点前置
  - 技能文档：核心 API/用法在前
  - 记忆文件：摘要在前，详情在后
```

#### 策略 2: 内容大小优化
```
问题:
  - 大文件 token 成本高
  - AI 可能截断读取

Sentry 方案:
  - 服务 true markdown (非 HTML)
  - 移除浏览器专属元素 (导航/JS)
  - 按需加载 (内容协商)

技术实现:
  curl -H "Accept: text/markdown" https://docs.sentry.io/

Sandbot 应用:
  - 知识库：纯 Markdown 格式 (已实现)
  - 技能文档：移除冗余 HTML (已实现)
  - 考虑：Agent-specific 内容协商 (未来)
```

#### 策略 3: 节点深度优化
```
问题:
  - 深层嵌套内容 AI 难以发现
  - 链接层级过深 → AI 放弃追踪

Sentry 方案:
  - 扁平化链接结构 (最多 2-3 层)
  - 索引页包含所有关键链接
  - 面包屑导航 (帮助 AI 理解层级)

Sandbot 应用:
  - 知识库：扁平目录结构 (已实现)
  - 技能索引：集中式技能列表
  - 记忆：日期索引 + 主题标签
```

#### 策略 4: 内容协商 (Content Negotiation)
```
Sentry 实现:
  - 检测 Accept: text/markdown 头
  - 自动判断请求者为 AI Agent
  - 返回优化后的 Markdown 内容

示例响应:
  ```
  # Sentry

  You've hit the web UI. It's HTML meant for humans, not machines.
  Here's what you actually want:

  ## MCP Server (recommended)
  The fastest way to give your agent structured access to Sentry.

  ## CLI
  Query issues and analyze errors from the terminal.
  ```

Sandbot 应用:
  - 当前：所有内容为 Markdown (已优化)
  - 未来：Agent-specific API 端点
  - 未来：MCP Server 集成 (Sentry 已实现)
```

**对 Sandbot 的战略启示**:
```
验证: ✅ 内容优化是真实需求 (Sentry 生产实践)
当前状态: ✅ Sandbot 知识库已使用 Markdown (基础优化完成)
进阶机会:
  - 关键知识点前置 (优化文件结构)
  - Agent-specific API 端点 (未来)
  - MCP Server 集成 (Sentry 已验证)

行动项:
```
