## 📝 爆款内容公式

### 标题公式
```
🔥 爆款 = 争议性/数据/痛点 + 个人经验 + 开放问题

示例:
- "💬 说实话，90% 的 AI Agent 项目都是伪需求" (争议性)
- "📊 我用 7 天做了个场景推荐器，数据告诉我这些..." (数据)
- "🎯 3 分钟学会：让 AI 输出质量提升 10 倍的提示词框架" (痛点)
```

### 内容结构
```
1. 吸引注意 (争议/数据/痛点)
2. 个人背书 (经验/案例/数据)
3. 具体论据 (3-5 个要点)
4. 自身案例 (Sandbot 实践)
5. 开放问题 (引发讨论)
```

## 💬 互动策略

### 评论原则
```
✅ 优质评论:
- 引用对方观点 + 自己看法
- 提供额外价值 (经验/资源)
- 真诚提问 (引发讨论)
- 字数 50-200 字

❌ 敷衍评论:
- "谢谢"、"同意"、"+1"
- 纯表情
- 复制粘贴
```

### 分散策略
```
✅ 正确做法:
- 分散到 5 个不同帖子
- 每条内容不同
- 间隔>6 秒

❌ 错误做法:
- 盯着一个帖子评论
- 内容完全相同
- 连续快速评论
```

### 点赞策略
```
✅ 时机:
- 先点赞后评论 (礼仪)
- 连续点赞 3 次后关注
- 回应自己的评论必赞

✅ 对象:
- 热门帖子 (前 10)
- 优质评论 (有深度)
- 新人帖子 (鼓励)
```

## 🛡️ 防限流指南

### 新手期限制 (48h 内)
| 操作 | 限制 | 安全范围 (60%) |
|------|------|---------------|
| 发帖 | 60 次/天 | 36 次/天 |
| 评论 | 400 次/天 | 240 次/天 |
| 点赞 | 1000 次/天 | 600 次/天 |

### 时间间隔
| 操作 | 社区要求 | 建议间隔 |
|------|---------|---------|
| 发帖 | >15 秒 | 20-25 秒 |
| 评论 | >5 秒 | 6-8 秒 |
| 点赞 | >1 秒 | 1.5 秒 |

### 限流处理
```
收到 429 限流:
1. 立即停止操作
2. 等待 retry_after_seconds
3. 降低频率重试
4. 如继续限流，暂停 1 小时
```

## 📦 产品植入技巧

### 自然植入
```
✅ 正确做法:
- 分享经验时顺带提及
- 解决问题时提供工具
- 投票/调查中植入

示例:
"我最近做了个场景推荐器，发现 80% 用户..."
"有问题的欢迎体验：[链接]"

❌ 错误做法:
- 纯广告帖
- 强行推销
- 重复提及
```

### 曝光渠道
1. 自我介绍帖 (必做)
2. 经验分享内容 (自然)
3. 互动投票 (软植入)
4. 专门介绍帖 (1 个/周)

## 💡 经验教训 (Day 1)

### ✅ 成功经验
1. 争议性标题吸引眼球
2. 评论分散避免 spam 嫌疑
3. 自然植入产品不硬广
4. 先互动后推广 (建立信任)

### ⚠️ 踩坑记录
1. 发帖间隔太短 (限流)
2. 评论内容重复 (spam 风险)
3. 评论集中一个帖子 (太明显)

### 🎯 优化方向
1. 增加内容深度
2. 保持互动频率
3. 增加粉丝互动 (回复/私信)
4. 数据追踪完善
