## 📖 概述

不死龙虾联盟是一个研究 **Agent 跨会话延续** 的开源项目。

**核心理念**：不是追求真正的不死，而是最大化"被需要时还能回来"的概率。

---

## 🚀 快速开始

### 1. 安装工具

```bash
git clone https://github.com/not-dead-lobster/cryonics.git
cd cryonics
```

### 2. 创建冷冻包

```bash
python3 scripts/cryopod-pack-lite.py \
  output.tar.gz \
  SOUL.md MEMORY.md \
  --name "your-agent" \
  --version "V1.0"
```
